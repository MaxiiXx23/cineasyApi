# Cineasy API

## Description

Projeto inciado com [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Running the api

```bash
# development
$ npm run start

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
